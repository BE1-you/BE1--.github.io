第一部分：HTML（页面结构）
1.1 HTML 基本结构
html
预览

  页面内容
</body>
</html>

第二部分：CSS（页面样式）
2.1 CSS 三种引入方式
1）行内样式（不推荐）
html
预览
<p style="color:red; font-size:20px">文字</p>
2）内部样式
html
预览
<style>
  p {
    color: red;
  }
</style>
3）外部样式（企业标准）
html
预览
<link rel="stylesheet" href="style.css">
2.2 CSS 语法
css
选择器 {
  属性: 值;
}
2.3 常用选择器
基础选择器
css
* {}                 /* 通配符 */
div {}               /* 标签选择器 */
.box {}              /* 类选择器（最常用） */
#header {}           /* id选择器 */
复合选择器
css
div p {}        /* 后代所有p */
div > p {}      /* 直接子元素p */
a:hover {}      /* 鼠标悬停 */
2.4 选择器优先级
!important > 行内 > id > 类 / 伪类 > 标签 > 通配符
2.5 常用样式（必背）
文字样式
css
color: red;              /* 颜色 */
font-size: 16px;          /* 大小 */
font-weight: bold;        /* 加粗 */
text-align: center;       /* 水平居中 */
line-height: 1.5;          /* 行高 */
text-decoration: none;    /* 去掉下划线 */
背景
css
background-color: #fff;
background-image: url(xxx.jpg);
background-size: cover;
盒子模型（核心）
css
width: 200px;
height: 200px;
padding: 10px;    /* 内边距 */
border: 1px solid #000; /* 边框 */
margin: 20px;     /* 外边距 */
box-sizing: border-box; /* 不会撑大盒子 */
居中
css
/* 块级元素水平居中 */
margin: 0 auto;

/* 文字垂直居中 */
line-height: 盒子高度;
浮动
css
float: left;
float: right;
clear: both; /* 清除浮动 */
定位
css
position: relative; /* 相对定位 */
position: absolute; /* 绝对定位 */
position: fixed;    /* 固定定位 */

top: 0;
left: 0;
弹性布局（Flex 必学）
css
display: flex;
justify-content: center; /* 水平居中 */
align-items: center;     /* 垂直居中 */
flex-wrap: wrap;         /* 换行 */
第三部分：JavaScript（交互逻辑）
3.1 JS 三种写法
标签内
<script> 标签
外部 .js 文件
3.2 变量
js
let name = '张三';
const age = 18; // 常量
3.3 数据类型
字符串 ''
数字 123
布尔 true/false
数组 []
对象 {}
3.4 运算符
js
+ - * / %
++ --
> < >= <= == ===
3.5 条件语句
js
if (age >= 18) {
  console.log('成年');
} else {
  console.log('未成年');
}
3.6 循环
js
for (let i = 0; i < 10; i++) {
  console.log(i);
}
3.7 函数
js
function fn() {
  return 123;
}
fn();
3.8 DOM 操作（最常用）
获取元素
js
document.getElementById('id');
document.querySelector('.box');
document.querySelectorAll('div');
操作内容
js
innerHTML = '<h1>标题</h1>';
innerText = '文字';
操作样式
js
style.color = 'red';
style.fontSize = '20px';
操作类名
js
classList.add('active');
classList.remove('active');
classList.toggle('active');
事件
js
btn.onclick = function() {}
btn.addEventListener('click', function() {})
常见事件：
click 点击
mouseenter 鼠标进入
input 输入
submit 表单提交
第四部分：学习路线（给新手的规划）
阶段 1：HTML + CSS
写出结构规范、语义化页面
会用 Flex 布局
能做：导航、卡片、登录页、个人主页
阶段 2：JavaScript 基础
变量、判断、循环、函数
DOM 操作、事件
能做：轮播、tab 切换、表单验证
阶段 3：前端工程化 & 框架
npm / git / vite
Vue 或 React
接口、Axios
