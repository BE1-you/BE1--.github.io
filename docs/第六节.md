************************** --> <div class="form-section"> <h2>⚠️ 表单属性易错点提醒</h2> <ul> <li>控件无name属性：提交时不会传递该控件的数据（最常见错误）</li> <li>disabled vs readonly：disabled禁用（不提交数据），readonly只读（提交数据）</li> <li>文件上传忘改enctype：form需设置enctype="multipart/form-data"，否则文件无法提交</li> <li>radio互斥失效：name属性不一致，导致可多选（需保证同组radio的name相同）</li> <li>required验证失效：form加了novalidate属性，禁用了浏览器默认验证</li> <li>textarea写value属性：textarea的初始值应写在标签内部，而非value属性</li> </ul> </div> </body> </html>
